Additional features:



1. Toggle 'O': enable snow effect.

2. Toggle 'D': enable wind effect on snow.  



3. Toggle 'L': enable lightning effect. Lightning can be seen all over the scene and they
 appear randomly.



4. Toggle 'R': enable rainbow effect. Rainbow is set on just one position (80, 0.0, 0.0)



5. Toggle 'G': enable bombing effect. Bomb will be dropped from current plane position.This is done with 
   collison detection. When the bomb is on ground, it should explode,
   please try a few times as under 
   some circumstance the bomb is not exploded. But
 generally it works perfectly fine.
